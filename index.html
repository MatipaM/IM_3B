Studio Ghibli Data Visualisation Website

Welcome to Matipa Marufu's data visualisation website. This website will be based on the Studio Ghibli API. It will contain a section in the data art and data visualisation pages which will allow the user to select their favourite movie and in turn the page will generate corresponding information in a digestable yet aesthetic format to the user.
